# Next.js Project Setup Guide

## Clone the Repository
```bash
git clone https://github.com/abdur-cader/312-dashboard
```


## Change Directory to the Project
```bash
cd 312-dashboard
```

## Install Dependencies
Run this to grab all needed packages:
```bash
npm install
```

**Requirements:** Node.js and npm installed on your system


## Start Development Server
Kick off the Next.js dev server:
```bash
npm run dev
```
If you want to start the server on a different port, use:
```
npm run dev -- -p [port number]
```


## View in Browser
View in http://localhost:3000 or your specified port on localhost.
